first copy this directory to the web server.
connect via ssh, change to this directory and do initial checkout:
 - cd /var/www/html/iris-reasoner_org/ant
 - cvs -d :pserver:anonymous@iris-reasoner.cvs.sourceforge.net:/cvsroot/iris-reasoner co iris
set appropriate permissions:
 - chmod ug+x snapshot.sh




the whole system is based on a cronjob:
-do "cronjob -e" 
-add "0 5 * * * /var/www/html/iris-reasoner_org/ant/snapshot.sh"
